wxsummary.csv is generated by scripts/wxsummary.py

It contains one row per year, state, and observation type. The columns:

type - observation type
  prcp is precipitation in millimeters
  tmin is the minimum observed temperature
  tmax is the maxmimum observed temperature
  tavg is the average observed temperature

state is the two-letter state code
year is obvious. the data is extracted for 1990-present

the following 12 columns are one sample per month in the bucket
of the first three columns

the data is averaged over all stations in the given state. missing fields
are filled in by simple interpolation of the neighboring months.
